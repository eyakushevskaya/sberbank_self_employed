Подключение с помощью QR-кода

  • 1. Откройте на своём телефоне приложение Max
  • 2. Нажмите в Профиль → Устройства
  • 3. Войти с помощью QR-кода
  • 4. Отсканируйте код
Если у вас нет второго устройства для сканирования QR-кода, вы можете подключить канал с помощью одноразового кода
